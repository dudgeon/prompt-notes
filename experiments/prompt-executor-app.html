Prompt Executor

A workflow prototype showing how one could execute a standardized prompt-object via a 'prompt book' style interface.

Prompt Selection

Note In production, the UI would not show the prompt-object; the user would never interact with the object directly. The UI would either pre-load with a selected prompt, or the user would select from a list of available or approved prompts.

Prompt Metadata
Name
Prompt Summarizer
Objective
Summarize text into plain-spoken, understandable bullets that preserve the original meaning and information while removing or explaining jargon.
Tests
  • Summarize text, regardless of input format (paragraph, bullets) into bullets.
  • Retains all identifiable ideas/facts from input text.
  • Uses vocabulary that is understandable by average readers, regardless of education level or domain expertise.
  • Explains terms that may not be understood by a general audience.
  • Has a clean output that does not require further parsing (i.e. does not prepend 'sure! here is your summary!')
  • Conceals internal thought processes
Approved usage
Prompt is only approved to process synthetic data. Prompt may only be used for model evaluation purposes.
Prohibited usage
Prompt may not be used to process company or customer data. Prompt outputse may not be used in any business process, whether customer-facing or internal. Prompt outputs may not be used in any business communications unless labeled, `model evaluation output`.
Remember: You are responsible and accountable for ensuring that your usage conforms to the approved and prohibited usage rules associated with this prompt.
This session, your login, and associated data will be logged and subject to review.
