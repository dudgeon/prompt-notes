Prompt Executor

A demo/paper prototype of an execution GUI for a standardized prompt-object.

Step 1: User selects prompt

User identifies the location of desired prompt, formatted in standard prompt-object format.

Prompt Selection
Note In production, the UI would not show the prompt-object; the user would never interact with the object directly. The UI would either pre-load with a selected prompt, or the user would select from a list of available or approved prompts.

Step 2: User reviews prompt metadata

Prompt executor displays metadata associated with the prompt object, including guidance on usage, objective, and tests.

User asserts that their usage conforms to the approved and permissibale usage rules associated with the prompt.
